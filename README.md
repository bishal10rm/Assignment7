KeenKeeper

KeenKeeper is a friendship management web application that helps users maintain meaningful relationships. Users can track friends, log interactions (Call, Text, Video), view interaction history through a timeline, and analyze communication patterns with visual analytics.

Technologies Used
React.js
React Router
Context API
Tailwind CSS
DaisyUI
React Icons
React Toastify
Recharts
Vite

Key Features
1. Friend Management:
View detailed friend profiles including contact information, relationship goals, interaction status, and personal notes.

2. Interaction Timeline:
Log Call, Text, and Video interactions directly from the Friend Details page. All activities are automatically stored and displayed on a dedicated Timeline page with filtering options.

3. Friendship Analytics:
Visualize communication patterns using interactive pie charts that show the number of Call, Text, and Video interactions.
